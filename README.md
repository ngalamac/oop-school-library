# 📖 School Library <a name="Morse-decoder"></a>

This project showcases the power of OOP concepts such as encapsulation, inheritance, and polymorphism in creating an organized and efficient library management system using the Ruby programming language. Explore the code to learn how OOP can be applied to real-world scenarios in Ruby and enhance your understanding of software development best practices in this context.

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:

- You need to have RUBY installed in your system.
- A Code Editor (Notepad++, VSCode, etc) or Windows command prompt(if you are on windows OS).

### Setup

Clone this repository to your desired folder:

```
git clone https://github.com/ngalamac/oop-school-library.git

cd oop-school-library
```
